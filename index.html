Tab Importer

Get a comma seperated list from a column in a tab or comma seperated file

Primarily used for getting a list of itemIDs from an exported catapult Worksheet. Though it could have other uses and has been working reliably for me for some time. Being developed for extended usage, including large file support.

üìù Not recommended for files with more than a thousand lines - maybe two or three, you can get away with 10 or more if you don't mind the unresponsiveness.

Try it

  • Drop Text File
  • Toggle Comma/Tab Button to choose data separator
  • Toggle AutoTrim to attempt removing header and footer rows
  • Adjust header and footer length with triangle buttons, reset with AutoTrim
  • Double Click Column Header to produce comma seperated list in lower pane
  • Triple Click comma seperated list to select
  • Copy to clipboard
