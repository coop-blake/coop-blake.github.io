Collection of tools used in POS tasks

Listed by assumed descending usefulness.

Windows Command Line ODBC Utilities

A set of command line tools for querying an ODBC data source and exporting the results to a Google Sheet or saving them in a local text file.

Used to move current inventory data for alternative access, checks and reporting

Tab Importer

Single page web application for quickly creating comma separated lists from a column of a tab or comma separated value text file.

Used for getting a list of item identifiers from exported Catapult worksheets, which can then be used for search and filtering.

UNFI Tools

Single page web applications for searching, visualizing, and comparing UNFI received data with an Inventory Text File.

The windows command line utilities can be used to produce an inventory file.
